# LEI-POO-Projeto
Projeto para a unidade curricular de Programação Orientada a Objetos.

## Compile

cmake -S . -B build -G "MinGW Makefiles" -DCMAKE_BUILD_TYPE=Debug

cmake --build build --parallel 4
